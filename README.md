# Cyvneil Enriquez — Full Stack Developer Portfolio

A premium, highly interactive, space-themed developer portfolio showcase. This site features customized canvas particles, real-time interactive physics models, customized scene telemetry configuration, and fluid animations.

---

## 🚀 Key Features

- **Interactive Starfield Canvas**: Dynamic background star particle simulation with custom theme presets, twinkle speeds, star densities, and responsive cursor parallax.
- **Custom Interactive Scene Playground**: Collapse-expandable UI panel allowing real-time personalization of the site's environment (Theme, Star density, Speed, Parallax Strength, and customized Vimeo/Mixkit video backdrop overrides).
- **Quantum Orbital Gyroscope Canvas**: Interactive, draggable 3D gyroscope displaying the core developer tech stack in the hero/skills sections.
- **Interactive Projects Ledger**: Cyber-telemetry lists displaying selected production work with full HUD-style project inspection drawers.
- **My Journey Timeline**: A fluid, responsive alternating milestone log animated using GSAP ScrollTrigger, featuring custom layouts for mobile/tablet resolutions.
- **Star Cursor Trails**: Multi-colored responsive cursor particle emitter.

## ⚡ Getting Started (Run Locally)

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- `npm` or `yarn`

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Cyv0712/cyvneil-portfolio-final.git
   cd cyvneil-portfolio-final
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open `http://localhost:3001` (or the configured port) in your browser.

4. **Build for production:**
   ```bash
   npm run build
   ```
